📝 Note Saver

Instalar

Salve suas notas rápidas direto no navegador

Com o Note Saver, você pode anotar ideias e lembretes sem precisar abrir outro app. Suas notas ficam armazenadas no próprio navegador, mesmo depois de fechar o Chrome.

✨ Funcionalidades

📝 Nota rápida

Escreva e salve uma nota instantaneamente no popup da extensão.

📋 Lista Infinita

Mantenha um histórico organizado com infinitas notas.

💾 Armazenamento local

As notas ficam salvas no navegador usando chrome.storage.local.

🧹 Limpar nota

Apague rapidamente o conteúdo atual com apenas um clique.

⚙️ Como instalar

  1. Baixe o projeto clicando em Instalar ou no GitHub.
  2. Extraia os arquivos em uma pasta no seu computador.
  3. No Chrome, acesse chrome://extensions/.
  4. Ative o Modo do desenvolvedor no canto superior direito.
  5. Clique em Carregar sem compactação e selecione a pasta do projeto.
  6. Pronto! A extensão aparecerá na barra do navegador.
